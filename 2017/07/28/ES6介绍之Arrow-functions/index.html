ES6介绍之Arrow-functions

React中许多地方都用到了ES6的新特性，Arrow function是其中之一，这里按照MDN上的介绍，自己总结翻译了一下。

简介

Arraw function expression 比function有更简介的语法，它不绑定自己的this,arguments,super,new.target，最适合用作非方法的函数，不能够用做构造函数。
