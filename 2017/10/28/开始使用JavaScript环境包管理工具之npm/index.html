开始使用JavaScript环境包管理工具之npm

Oct 28, 2017

什么是npm

npm 是一个js包管理工具，可以帮助分享你的代码、使用别人分享的代码，并且提供简单的方式管理不同版本的代码。

安装Node.js和更新npm

使用npm工具需要首先安装Node.js环境，Node环境可以到官方网站下载，下载后环境内就可以使用npm命令管理js包代码。
