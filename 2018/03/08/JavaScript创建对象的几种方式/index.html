JavaScript创建对象的几种方式

Mar 8, 2018

对象字面量

缺点是使用同一接口创建很多对象，会产生大量重复的代码。

工厂模式

缺点是不能识别对象的类型(即怎样知道一个对象的类型，即使用instanceof识别)。

构造函数模式

这里在复习一下使用new操作符后的执行过程：

  1. 创建一个新对象。
  2. 将构造函数作用域执行新对象（this指向了新对象）。
  3. 执行构造函数中的代码。
  4. 返回新对象。

构造函数中的缺点就是，当创建的对象中包含Function时，实例上的同名函数是不相等的。

组合构造函数模式和原型模式
