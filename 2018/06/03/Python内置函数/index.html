正则

正则表达式是可以匹配文本片段的模式。

正则需要注意的几点：

  1. 使用原生字符串如：r’python\.org’，避免多次使用’\\.’让re模块转义和解释器两次转义。
  2. re.match()函数在字符串的开始处匹配模式，如果想匹配整个字符串，可以在结尾家美元符。
  3. 正则默认为贪婪模式，可以在子模式后加?改为非贪婪模式，如：pattern = r’\\(.+?)\\‘。

集合

初始化，集合是由序列（或者其他可迭代的对象）构建的。

性质

  • 集合中元素的顺序是随意的。
  • 集合是可变的，不能用做字典中的健。
  • 集合只能包含不可变的值，所以不能包含其他集合。
