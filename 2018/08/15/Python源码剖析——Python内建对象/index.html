前言

在Python的世界中，一切都是对象，包括面向对象中的“类”也是通过Python内的对象来实现的。

Python内的对象

在Python中，对象就是C的结构体在堆上申请的一块内存。

  • Python中所有的内建的类型对象(int, str等对象)都是被静态初始化的。
  • 对象一旦创建, 内存大小就是不变的(变长的对象内部维护一个指向可变内存的指针)
