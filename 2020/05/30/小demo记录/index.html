解决安装 MySQLdb 问题

  1. pip3 install PyMySQL
  2. 修改 pymsql/init.py 中的内容，调用 install_as_MySQLdb() 函数
  3. 在 python 的 site-package 包中创建 MySQLdb.py
  4. 在 MySQLdb.py 中 import pymysql
